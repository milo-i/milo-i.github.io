HTML Table Reference

Table tags

Tag Name Description
<h1> Heading 1 The h1 is an HTML tag that indicates a heading on a website
<p> Paragraph The p tag defines a paragraph
<form> Form The form tag is used to create an HTML form for user input.
<img> Img The img tag is used to embed an image in an HTML page.
<div> Div The div tag defines a division or a section in an HTML document.
<table> Table The table tag defines an HTML table.

Table Attributes

Attribute Name Description
src Source The purpose of the HTML src attribute is to specify a URI for an external file or resource.
colspan Column Span Defines how many columns a td element should span
href Href The href attribute specifies the URL of the page the link goes to
alt Alternate specifies an alternate text for an image, if the image for some reason cannot be displayed.
